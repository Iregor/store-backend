# store-backend


### System requirements:
JDK 17 amazon corretto  
IntellijIdea

### Startup instructions:
1. Download zip-file
2. Unpack zip-file
3. Open app in IntellijIdea
4. Use you PostgresDB properties in config and docker-compose.yml:
    - POSTGRES_USER
    - POSTGRES_PASSWORD
5. mvn clean package
6. run app from main class or from docker-compose.yml
